# Square Version

Current official version: **v0.3.0**

## Why v0.3.0

Square has moved beyond the original v0.1 baseline. The app now includes basic interaction/actions, device-specific layouts, image content, group movement, and a function-mode logic overlay.

## Version Map

| Version | Meaning |
| --- | --- |
| v0.1 | Core Book/Page/Square canvas engine |
| v0.2 | Basic Square Actions and Edit/Run interaction model |
| v0.3 | Current app: device layouts, image content, group movement, logic overlay, Square terminology |

## Compatibility Note

The localStorage key may still contain `v0.1` for backward compatibility with existing browser data. This is storage compatibility, not the official app version.
